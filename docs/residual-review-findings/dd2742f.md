## Reliability / scale
1. **(P2) Per-request full re-import.** Every `/api/platform/*` drilldown calls `_sync_store()`,
   which re-reads and re-parses the entire trace corpus (`load_jsonl_with_health` →
   `read_text().splitlines()` per file) on every request. `import_checkpoints`
   (`sqlite_store.py`) is written by `record_checkpoint` but **never read**, so import is not
   incremental. Fix: gate import on file mtime/offset via the checkpoint table; stream lines;
   single-flight concurrent rebuilds. Files: `gateway/app.py` `_sync_store`,
   `platform/importers.py`, `platform/evidence.py` `load_jsonl_with_health`.
2. **(P2) No stale-on-failure fallback.** `SnapshotCache.get()` (`snapshots.py`) and the
   drilldown `_sync_store()` calls are unguarded — a transient SQLite lock makes `GET /`,
   `/api/platform/overview`, and the drilldowns return 500 instead of serving the last good
   (stale-labelled) snapshot. Fix: try/except around the builder/sync; serve cached + STALE on
   failure with a short cooldown; re-raise only when no cache exists. (Guard path is unaffected.)
3. **(residual) No `busy_timeout`.** Neither `SqliteEvidenceStore._connect` nor
   `CanaryVault._connect` sets `PRAGMA busy_timeout`; `SnapshotCache` mutates `_cached`
   lock-free. Fine for single-process local use; revisit if a multi-worker deploy is added.

## Truthfulness
4. **(P2/P3) Filtered overview mixes scopes.** `GET /api/platform/overview?session_id=X` returns
   a filtered `total`/`recent` but **global** `by_action`/`by_phase`/`detector_hits`
   (`sqlite_store.py` `decision_overview`/`cift_overview`/`canary_overview` run unfiltered
   `GROUP BY`), so `total != sum(by_action)`. The default (unfiltered, cached) overview is
   consistent. Fix: either scope the aggregates with `_event_where(query)`, or make the overview
   always-global by dropping `session_id`/`offset` from the overview endpoint (use drilldowns to
   filter) and update `test_overview_snapshot_is_cached_within_window`.
5. **(P3) Session nimbus score under-reports.** `sessions()` takes the latest event's
   `nimbus_score`; a trailing canary-plant event (nimbus 0) zeroes the session's
   `nimbus_cumulative_score`. Fix: `MAX(nimbus_score) OVER (PARTITION BY session_id)` (the ledger
   is monotonic). `sqlite_store.py` `sessions()`.

## Robustness (P3)
6. **Wall-clock freeze.** `SnapshotCache` uses `time.time`; a backward NTP step makes
   `now - _generated_at` negative so it never refreshes and shows `cached · -Ns`. Fix:
   `time.monotonic()` for age/refresh, keep wall-clock only for the displayed `generated_at`;
   clamp age ≥ 0. `snapshots.py`.
7. **Wrong-but-valid key misdiagnosis.** A valid-but-wrong vault key yields N per-row
   `CORRUPT_ROW` warnings (no single "key mismatch" signal), inviting an operator to delete a
   **recoverable** vault. Fix: when most/all rows fail to decrypt, collapse to one DEGRADED
   "key does not match stored tokens (recoverable — do not delete)" and cap per-row warnings.
   `canaries.py` `restore`/`health_warnings`.
8. **metrics.json type crash.** A structurally-valid `metrics.json` with a non-numeric metric
   value (or `success_criteria` as a list) 500s the dashboard via `_kpis`/`_criteria` format
   strings. Fix: coerce/guard value types at the render seam. `dashboard/render.py`.

## API / minor (P3)
9. **Unknown `?format=` silently returns JSON** (`gateway/app.py` `platform_export`). Consider a
   400 allowlist, or document the JSON default.
10. **`schema_version` duplicated** at top-level `PlatformOverview` and in `SnapshotMeta`
    (`store.py`). Drop it from `SnapshotMeta` (it's freshness metadata, not an API version).
11. **`render_html` signature is a breaking change** for any external importer (internal callers
    updated). Accepted; note in a changelog if `aegis.dashboard` is ever published.
12. **(P3, pre-existing) Import re-redaction is pattern-only.** The importer re-redacts summaries
    with `redact_text` (credential regexes) but not broker-known non-pattern secrets or
    non-pattern canary formats written by a registry that doesn't know them. Bounded impact
    (default fake store empty; canaries synthetic). Fix at the write seam
    (`client._build_event` → `broker.redacted_text`).
